# Balkazar website

Official bilingual booking website for Balkazar, published with GitHub Pages at
[balkazar.org](https://balkazar.org).

## Key pages

- `/` — English booking site
- `/zh/` — Taiwan Traditional Chinese booking site
- `/zh/weiya/` — 尾牙／春酒 advertising landing page
- `/epk/` and `/zh/epk/` — electronic press kits
- `/gallery/` and `/zh/gallery/` — photo galleries
